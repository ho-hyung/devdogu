**24 browser-only developer tools. No data sent to servers.**

---

## Why DevDogu?

- **Privacy first** - All tools run 100% client-side. Your data never leaves the browser.
- **All-in-one** - 24 essential dev tools in one place. No more bookmarking 20 different sites.
- **Fast** - Static site, loads instantly. No spinners, no waiting.
- **Multilingual** - Available in Korean, English, Japanese, and Chinese.
- **Open source** - MIT licensed. Contribute or fork freely.

---

## Tech Stack

- **Framework**: Next.js 14 (App Router, static export)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Pretendard (Korean) + JetBrains Mono (code)
- **Deploy**: Vercel
- **i18n**: Custom implementation (path-based, minimal bundle)

## Getting Started

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # Static build
```
