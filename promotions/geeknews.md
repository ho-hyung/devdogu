# GeekNews 게시글 초안

## 제목
DevDogu — 서버 전송 없이 브라우저에서만 동작하는 개발자 도구 23개 모음

## 본문

개발하면서 자주 쓰는 도구들을 한곳에 모았습니다.

**사이트**: https://devdogu.vercel.app

### 주요 특징

- **100% 클라이언트 사이드** — 입력한 데이터가 서버로 전송되지 않습니다. 회사 코드나 JWT 토큰도 안심하고 붙여넣기 할 수 있습니다.
- **23개 도구** — JSON 포매터, Base64, JWT 디코더, Diff 비교기, Cron 빌더, 해시 생성기, QR 생성기, SQL 포매터 등
- **치트시트** — Git, Docker, Linux, HTTP 상태코드, 정규식, Cron 문법 정리
- **한국어 UI** — 한국어로 된 개발자 도구 모음이 거의 없어서 직접 만들었습니다

### 기술 스택

- Next.js 14 (App Router, 정적 빌드)
- TypeScript + Tailwind CSS
- Vercel 배포 (정적 사이트이므로 비용 0원)

### 만든 이유

JSON 포매터 쓸 때마다 매번 다른 사이트 찾아다니는 게 번거로웠고, 특히 "이 사이트가 내 데이터를 서버로 보내는 건 아닌지" 걱정이 됐습니다. 그래서 완전히 브라우저에서만 동작하는 도구 모음을 만들었습니다.

소스코드도 공개되어 있어서 직접 확인하실 수 있습니다.
