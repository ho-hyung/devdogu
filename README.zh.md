# DevDogu

[한국어](README.md) | [English](README.en.md) | [日本語](README.ja.md)

**24个纯浏览器端开发者工具。数据不会发送到服务器。**

[devdogu.vercel.app](https://devdogu.vercel.app) | [报告Bug](https://github.com/your-username/devdogu/issues)

---

## 为什么选择DevDogu？

- **隐私优先** - 所有工具100%在客户端运行。数据不会离开浏览器。
- **一站式** - 24个必备开发者工具集于一处。无需收藏20个不同的网站。
- **快速** - 静态站点，即时加载。没有加载动画，没有等待。
- **多语言** - 提供韩语、英语、日语和中文UI。
- **开源** - MIT许可证。自由贡献或fork。

---

## 技术栈

- **框架**: Next.js 14 (App Router, static export)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **字体**: Pretendard (韩语) + JetBrains Mono (代码)
- **部署**: Vercel
- **i18n**: 自定义实现 (基于路径，最小化打包体积)

## 快速开始

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # 静态构建
```
